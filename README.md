## 🚀 Getting Started

1. **Clone the repository**
   ```
   git clone https://github.com/gaureshpai/venjix.git
   ```

2. **Install dependencies**
   ```
   npm install
   ```

3. **Run the development server**
   ```
   npm run dev
   ```

4. **Open your browser**
   Navigate to [http://localhost:3000](http://localhost:3000) to see the magic! ✨
